# Notes V2

## Endpoints

### Files
* GET /files?search={search}&count={count}&start={start}
* POST /file/new
* GET /file/{id}
* PUT /file/{id}
* DELETE /file/{id}

### Tags
* GET /tags?search={search}&count={count}&start={start}

### Collections
* GET /collections?search={search}&count={count}&start={start}
* POST /collection/new
* PUT /collection/{id}
* DELETE /collection/{id}
